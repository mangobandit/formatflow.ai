Privacy Policy

Last updated: May 2026

FormatFlow helps users plan and review document translation workflows for PPTX and DOCX files.

Information Used

The FormatFlow Document QA action may use details you provide, such as document type, source language, target languages, audience, glossary notes and review notes.

How It Is Used

  • To create workflow plans.
  • To create QA checklists.
  • To suggest layout-risk checks and reviewer handoff notes.

Document Content

The current action is intended for workflow planning and QA support. Avoid submitting sensitive document content unless you are comfortable doing so.

Storage

The current action endpoints are designed to return a response and do not intentionally save workflow requests in a customer database.

Third Parties

FormatFlow uses hosting and platform services such as Vercel, GitHub and ChatGPT/OpenAI to operate the website and action experience.
